# Структура проекта file2text

## 📁 Организация файлов

```
file2text/
├── file2text/              # Основной пакет
│   ├── __init__.py         # Главный экспорт
│   ├── core/               # Основные модули
│   │   ├── transcriber.py  # Транскрипция (Whisper)
│   │   ├── diarizer.py     # Диаризация спикеров
│   │   ├── summarizer.py   # Суммаризация
│   │   ├── vectorizer.py   # Векторизация
│   │   └── file2text.py    # Главный класс
│   ├── utils/              # Утилиты
│   │   ├── audio_converter.py
│   │   ├── text_cleaner.py
│   │   └── config.py
│   └── cli/                # CLI интерфейс
│       └── main.py
│
├── examples/               # Примеры использования
│   ├── basic_usage.py      # Базовый пример
│   └── legacy/             # Старые скрипты
│       ├── main.py
│       ├── summarize.py
│       ├── logicCombrain.py
│       └── README.md
│
├── docs/                   # Документация
│   ├── ARCHITECTURE.md
│   ├── DEVELOPMENT_PLAN.md
│   ├── DISCUSSION.md
│   ├── IMPLEMENTATION_SUMMARY.md
│   └── MODULE_README.md
│
├── files/                  # Входные аудио файлы (игнорируется git)
├── text/                   # Результаты транскрипции (игнорируется git)
├── sumText/                # Результаты суммаризации (игнорируется git)
├── logical_blocks/         # Логические блоки (игнорируется git)
│
├── ffmpeg/                # FFmpeg бинарники
│
├── .env                    # Конфигурация (игнорируется git)
├── .env.example            # Пример конфигурации
├── .gitignore              # Git ignore правила
├── setup.py                # Установка пакета
└── README.md               # Главная документация
```

## 🗂️ Описание папок

### `file2text/` - Основной пакет
Содержит весь код библиотеки, организованный по модулям.

### `examples/` - Примеры
- `basic_usage.py` - Базовый пример использования
- `legacy/` - Старые скрипты для обратной совместимости

### `docs/` - Документация
Вся техническая документация проекта.

### `files/`, `text/`, `sumText/`, `logical_blocks/`
Рабочие папки с данными (игнорируются git).

### `ffmpeg/`
FFmpeg бинарники для конвертации аудио.
